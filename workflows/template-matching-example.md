# OpenCV 模板匹配工作流

## 步骤

1. 激活 PyCharm 窗口

2. 点击运行按钮（使用模板匹配）

   ```yaml
   operation: click_run_button
   parameters:
     template: run_button.png
     confidence: 0.8
   ```

3. 点击数据库按钮（使用模板匹配）

   ```yaml
   operation: click_database_button
   parameters:
     template: database-button.png
   ```
